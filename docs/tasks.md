# PMTscope — 项目开发任务列表

> 基于 `docs/inference.md` v2.1 需求文档制定

---

## 阶段一：项目初始化与环境搭建

| 编号 | 任务 | 说明 | 优先级 |
|------|------|------|--------|
| T1.1 | 初始化项目目录结构 | 按推荐结构创建 `app.py`, `data_loader.py`, `plots.py`, `utils.py`, `config.yaml`, `requirements.txt`, `README.md` | P0 |
| T1.2 | 编写 `requirements.txt` | 列出依赖：`streamlit`, `pandas`, `numpy`, `plotly`, `scipy`, `pyyaml` | P0 |
| T1.3 | 编写 `config.yaml` | 定义数据库路径、默认参数（bin 数量、离群点检测规则、中心值方法等） | P0 |
| T1.4 | 创建虚拟环境并安装依赖 | `python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt` | P0 |

---

## 阶段二：数据层（data_loader.py）

| 编号 | 任务 | 说明 | 优先级 |
|------|------|------|--------|
| T2.1 | 实现数据库/CSV 读取模块 | 从 `config.yaml` 读取路径，支持 SQLite 和 CSV 两种数据源，统一返回 `pandas.DataFrame` | P0 |
| T2.2 | 实现数据清洗逻辑 | 处理缺失值标记、类型转换、时间字段解析 | P0 |
| T2.3 | 实现全局过滤器 | 按 `hv` 范围、`run_id` 列表、`pmt_id` 列表、`run_type`/`run_tag`、时间范围过滤 DataFrame | P0 |
| T2.4 | 实现统计计算函数 | 计算中位数、均值、标准差、IQR，供离群点检测和中心值使用 | P0 |
| T2.5 | 实现离群点检测 | 支持 3σ 和 IQR 两种规则（可配置），返回离群点标记列 | P1 |
| T2.6 | 实现查询接口 | 按 `pmt_id`、`run_id` 或组合查询，返回完整记录子集 | P1 |

---

## 阶段三：可视化层（plots.py）

| 编号 | 任务 | 说明 | 优先级 |
|------|------|------|--------|
| T3.1 | 实现参数直方图（F2） | `spe_gain`、`dark_count_rate`、`after_pulse_probability` 三张直方图，支持动态 bin 数量、KDE 叠加、Plotly zoom/pan | P0 |
| T3.2 | 实现三维参数空间分布图（F3） | X/Y/Z 分别为 `spe_gain`、`dark_count_rate`、`after_pulse_probability`，跳过含缺失值记录，颜色按 `pmt_id` 或 `run_id` 映射，支持旋转缩放 | P0 |
| T3.3 | 实现 3D 图点击回调 | 点击数据点时返回该点完整字段信息，供信息面板展示 | P0 |
| T3.4 | 实现参数 vs. PMT ID 趋势散点图（F4） | 三张散点图：`spe_gain` / `dark_count_rate` / `after_pulse_probability` vs `pmt_id`，X 轴按数值/字典序排列，添加中心值水平虚线，自定义 hover tooltip，支持 zoom 和框选 | P0 |
| T3.5 | 实现离群点高亮渲染（F5） | 在直方图和趋势散点图中用红色标注离群点，趋势图离群点旁 `pmt_id` 标签高亮 | P1 |
| T3.6 | 实现多 Run 对比功能（F6） | 直方图支持半透明重叠/分组显示，3D 图不同 run 用不同颜色/符号，趋势图叠加不同 run 数据 | P1 |

## 阶段四：主应用与 UI（app.py）

| 编号 | 任务 | 说明 | 优先级 |
|------|------|------|--------|
| T4.1 | 搭建 Streamlit 页面骨架 | 设置页面标题、图标、布局模式（wide），加载配置和初始化数据 | P0 |
| T4.2 | 实现侧边栏全局过滤器（F1） | `hv` 范围滑块、`run_id` 多选下拉、`pmt_id` 多选下拉（支持搜索）、`run_type`/`run_tag` 可选筛选、时间范围选择器 | P0 |
| T4.3 | 实现过滤器联动刷新 | 任一过滤条件改变时，通过 `st.session_state` 触发所有图表同步更新 | P0 |
| T4.4 | 实现参数直方图页面区域（F2） | 嵌入三张直方图，bin 数量滑块控件，KDE 开关 | P0 |
| T4.5 | 实现 3D 散点图页面区域（F3） | 嵌入 3D 图，颜色映射下拉（`pmt_id` / `run_id`），右侧信息面板显示点击数据点详情 | P0 |
| T4.6 | 实现趋势散点图页面区域（F4） | 嵌入三张趋势散点图，中心值方法切换（中位数/均值），hover 信息展示 | P0 |
| T4.7 | 实现离群点预警控件（F5） | 离群点检测规则切换（3σ / IQR），开关离群点高亮 | P1 |
| T4.8 | 实现多 Run 对比控件（F6） | Run 多选控件，对比模式切换 | P1 |
| T4.9 | 实现快速查询面板（F7） | 输入框接受 `pmt_id` 或 `run_id`（支持逗号分隔组合查询），表格展示全部字段，支持排序、搜索、导出 CSV | P1 |
| T4.10 | 实现图表导出功能 | 利用 Plotly 内置工具栏提供 PNG 下载 | P0 |

## 优先级说明

| 级别 | 含义 |
|------|------|
| **P0** | 核心功能，必须首先完成，阻塞后续任务 |
| **P1** | 重要功能，在核心功能后实现 |
| **P2** | 辅助/锦上添花，时间允许时完成 |
